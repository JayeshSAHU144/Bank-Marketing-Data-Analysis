# 📊 Bank Marketing Data Analysis

## 📌 Overview
This project is part of my **Data Analytics Internship**, where I analyzed bank marketing data to understand customer behavior and predict whether a client will subscribe to a term deposit.

---

## 🎯 Objective
- Perform Exploratory Data Analysis (EDA)  
- Identify key factors influencing customer subscription  
- Build a predictive model using Logistic Regression  

---

## 🛠️ Tools & Technologies
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Scikit-learn  

---

## 📂 Dataset Features
- Age, Job, Marital Status  
- Education, Contact Type  
- Campaign Details  
- Call Duration  
- Previous Interactions  
- Target Variable: **Subscription (Yes/No)**  

---

## 🔍 Key Analysis
- Data cleaning and preprocessing  
- Customer demographics analysis  
- Campaign effectiveness analysis  
- Correlation heatmap  
- Feature importance  

---

## 📈 Key Insights
- Longer call duration increases chances of subscription  
- Fewer contact attempts lead to better conversion  
- Customer demographics influence subscription behavior  
- Dataset shows class imbalance  

---

## 🤖 Model Building
- Logistic Regression model was used  
- Achieved high accuracy (~90%)  
- Improved model using class balancing  

---

## 📊 Results
The model successfully predicts customer subscription and helps understand key factors influencing decisions.

---

## 📌 Conclusion
This project demonstrates how data analysis and machine learning can be used to improve marketing strategies and decision-making.

---
